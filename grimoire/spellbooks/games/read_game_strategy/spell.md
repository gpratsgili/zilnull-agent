---
name: "read_game_strategy"
description: "Read ZIL's accumulated strategic knowledge for a specific game."
---
Reads `spirits/zil/games/<game_id>/strategy.md` — ZIL's living strategic document for the game, updated after runs to reflect what it has learned.

**Charge:** free (inner spirit read)
**Widening required:** no

**Inputs:**
- `game_id` (string) — e.g. `"sts2"`

This document is ZIL's own synthesis, not an API reference. It contains what ZIL currently believes about how to play well — updated by rewriting, not by appending.
